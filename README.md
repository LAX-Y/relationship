你有没有过这样的时刻——

- 对着聊天框犹豫半天，不知道该回什么
- 反复琢磨一句话到底是什么意思
- 明明很喜欢，却总在关键时候说错话
- 不确定自己的感觉是喜欢，还是只是不甘心

这个工具就是为这些时刻准备的。它不是"恋爱攻略"，不教你话术，不帮你套路——它用心理学的眼光，帮你**看清自己**，**理解对方**，让每一次互动都在你清醒的选择中发生。

---

## 它能做什么

| 🧠 认识自己 | 👁 理解对方 | ✍️ 更好地表达 |
|:---:|:---:|:---:|
| 我现在的感觉是什么 | TA 的行为在说什么 | 怎么回消息不伤害 |
| 我的依恋模式在影响我吗 | 我们的互动健康吗 | 紧张时该怎么说话 |
| 值不值得继续走下去 | 谁是更需要这段关系的人 | 表达需求而非指责 |

## 安装

**OpenClaw：**
```bash
cd ~/.openclaw/workspace/skills
git clone https://github.com/LAX-Y/relationship.git relationship-coach-oss
openclaw gateway restart
```

**Claude Code / Cursor / ChatGPT：** 克隆后自动识别 `CLAUDE.md` 或 `AGENTS.md`。

## 理论基础

本工具的分析基于 **26 个经过实证检验的心理学理论**，涵盖依恋理论、戈特曼婚姻实验室研究、斯滕伯格爱情三角、非暴力沟通等。每个理论均标注原始论文出处。

完整理论手册见 `references/theory_reference.md`。
