## 它能做什么

把聊天记录发给它，它告诉你：

- 🩺 **互动健康吗？** — 戈特曼 5:1 正负比 + 四骑士检测
- 🔄 **你们是什么模式？** — 依恋风格诊断 + 焦虑-回避陷阱预警
- 🧠 **你的大脑在骗你吗？** — 认知失调、曝光效应、蔡格尼克效应
- 💔 **值不值得继续？** — 斯滕伯格三角 + Rusbult 投入模型
- ✍️ **怎么回消息？** — NVC 格式，3 种语气可选

> 不是通用的"好好沟通"——每条建议都有科学依据和置信度标注。

## 理论基础

26 个实证心理学理论，全部标注原始论文出处：

- **经典（1927-1968）：** 蔡格尼克、曝光效应、互惠喜欢、最少兴趣、心理抗拒
- **基础（1950s-1990s）：** 依恋理论、NVC、社会渗透、认知失调、情绪ABC
- **实验（1979-2015）：** 戈特曼四骑士、斯滕伯格三角、Rusbult 投入模型、峰终定律、损失厌恶
- **最新（2004-2017）：** 成长vs命运心态、关系动荡、感知回应性、感恩绑定、智慧推理

## Relationship Coach — Science-Backed Dating Advice

Paste your chat records. Get analysis powered by **26 empirically validated psychological theories** — not generic tips, but evidence-backed coaching with confidence annotations.

**Works on:** OpenClaw / Claude Code / Cursor / ChatGPT / any LLM.

### What It Analyzes

- **Interaction Health** — Gottman's 5:1 ratio + Four Horsemen
- **Attachment Style** — Bowlby & Bartholomew diagnosis
- **Power Dynamics** — Least Interest + Reactance
- **Communication** — NVC + Social Penetration + PPR + ACR
- **Cognitive Biases** — Dissonance, ABC, Zeigarnik, Growth/Destiny
- **Love Diagnosis** — Sternberg's Triangle + Rusbult's Investment Model

git clone https://github.com/LAX-Y/relationship.git
